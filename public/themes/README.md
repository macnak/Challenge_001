# Theme Asset Conventions

Place theme-level brand assets under `public/themes/<theme-pack>/`.

Recommended filenames:

- `logo.svg` (or `logo.png` / `logo.webp`)
- `watermark.svg` (or `watermark.png` / `watermark.webp`)

Example:

- `public/themes/inferno/logo.svg`
- `public/themes/inferno/watermark.svg`

Use with:

- `THEME_PACK=<theme-pack>`
- `BRAND_LOGO_MODE=theme`
- `THEME_WATERMARK=on` (optional)
