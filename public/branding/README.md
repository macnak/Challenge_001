# Branding Asset Conventions

Place white-label brand assets under the `public/branding/<tenant-id>/` directory.

Recommended filenames:

- `logo.svg` (or `logo.png` / `logo.webp`)
- `watermark.svg` (or `watermark.png` / `watermark.webp`)

Example:

- `public/branding/acme/logo.svg`
- `public/branding/acme/watermark.svg`

`TENANT_ID` should match the folder name (case-insensitive, slug-safe).
